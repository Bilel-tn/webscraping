Demo web Scraping TBG Camp

Le but de ce premier exemple est de vous montrer comment faire le web scraping à partir d'une page web basique.

  • La déclaration < DOCTYPE > représente le type de document, et aide les navigateurs à afficher correctement les pages Web. Elle ne doit apparaître qu'une seule fois, en haut de la page (avant toute balise HTML).
  • Les titres HTML sont définis à l'aide des balises <h1> à <h6> . <h1> définit le titre le plus important. <h6> définit l'en-tête le moins important.
  • Les paragraphes HTML sont définis à l'aide de la balise <p>.
  • Les liens HTML sont définis à l'aide de la balise <a>. Le lien de la page est spécifié à l'aide de l'attribut "href".
  • Les images HTML sont définies avec la balise <img>. Le fichier source (src), le texte alternatif (alt), la largeur et la hauteur sont fournis comme attributs.
